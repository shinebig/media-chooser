Media Chooser

  • Collect media from users with just a line of code.
  • Allow users to upload new media from their computer or choose from Facebook, Instagram, Flickr or Picasa.
  • Grab captions, dimensions, thumbnails and other info in the process.
  • Scale and size media on the fly.
  • Collect individual photos or entire albums.

Get started or try it now.

  1. Register for an account.

    2. We'll email your registration data.

  2. Setup Media Chooser.

    3. Include the library (jQuery required): 

    <script src="http://cdn.getchute.com/media-chooser.min.js"></script>
    Fill in your Application ID (sent via email): 
    Chute.setApp('application_id');

  3. Start collecting media.

    4. Chute.MediaChooser.choose(function(urls, data){
	// urls is an array of direct media URLs
	// data array contains additional information about media, like dimensions
});
